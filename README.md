# Texela

Texela is a simple, browser-based SaaS that converts images into editable text using OCR and allows users to export images as PDFs — fast, private, and easy to use.

---

## ✨ Features

- Upload images (JPG, PNG, etc.)
- Extract text from images using OCR
- Convert images into PDF files
- Preview images before processing
- Runs entirely in the browser
- No sign-up required

## 🛠 Tech Stack

- **Framework:** Next.js
- **Frontend:** React
- **OCR:** Tesseract.js
- **PDF:** pdf-lib
- **Deployment:** Vercel
